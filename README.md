# DNGExport
Blender add-on to export renders to Adobe DNG raw format.

See in [Blender extensions website](https://extensions.blender.org/approval-queue/export-render-as-dng/)

Thanks to [schoolpost](https://github.com/schoolpost) for his work on [PiDNG](https://github.com/schoolpost/PiDNG.git) python module.
